# Implementação

## Status

Aceito

## Context

Como protocolos temos os predominantes o Rest e o graphQl para apis json.

## Decision

Para manter a complexidade menor e seguindo melhor a linha de conhecimento e experiência dos membros do grupo, escolhermos seguir com o Rest.

## Consequences

Com o Rest temos uma implementação que está mais tempo no mercado e com mais tempo de uso do time de desenvolvimento o que facilita a produção do projeto.
