Cline


Projeto

Com o intuito de tornar a experiência dos nossos pacientes ainda mais agradável, o CLine gerencia e agenda de maneira simples e intuitiva nossas consultas e exames, fazendo com que toda a jornada de trabalho seja simplificada em apenas alguns cliques.

Este é uma aplicação web para gerenciamento de consultas e exames clinicos. A aplicação permite gerir os casos médicos e os diferentes utilizadores da aplicação (administrador, paciente, médico).


Bibliotecas e tecnologias utilizadas:

  • HTML
  • CSS
  • JavaScript
  • Node
  • TypeScript
  • Nest
  • Passport
  • Bcrypt
  • Prisma
  • JWT
